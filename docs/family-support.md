# 家庭支持模块框架 (Family Support Module)

## 概述

本文档定义了孕期教育网站的家庭支持模块框架，重点关注丈夫（伴侣）在孕期中的角色、沟通技巧和情绪支持要点。

**目标**：帮助家庭成员（尤其是丈夫）更好地支持孕期妻子  
**适用范围**：全阶段（孕早期、孕中期、孕晚期）

---

## 1. 丈夫支持角色定义

### 1.1 孕早期（1-12周）

| 支持领域 | 具体行动 | 说明 |
|----------|----------|------|
| 情感支持 | 倾听、安慰、陪伴 | 孕妇可能出现情绪波动，需要理解与耐心 |
| 生活照顾 | 准备健康饮食、陪同产检 | 一起面对早孕反应 |
| 信息支持 | 了解孕期知识、共同学习 | 成为妻子的"学习伙伴" |
| 决策支持 | 了解分娩选择、参与决策 | 共同规划未来 |

### 1.2 孕中期（13-27周）

| 支持领域 | 具体行动 | 说明 |
|----------|----------|------|
| 情感支持 | 鼓励自我关爱、赞美妻子 | 孕期是建立亲密关系的好时机 |
| 生活照顾 | 陪伴运动、共同准备宝宝用品 | 参与感增强 |
| 信息支持 | 参加产前课程、了解分娩知识 | 做好知识准备 |
| 亲子沟通 | 对着肚子说话、感受胎动 | 建立与宝宝的联系 |

### 1.3 孕晚期（28-40周）

| 支持领域 | 具体行动 | 说明 |
|----------|----------|------|
| 情感支持 | 缓解产前焦虑、给予信心 | 面对分娩的紧张情绪 |
| 生活照顾 | 照顾日常生活、准备待产包 | 做好分娩准备 |
| 陪产准备 | 了解陪产流程、准备陪产物品 | 准备迎接宝宝 |
| 产后支持 | 了解月子安排、育儿分工 | 规划产后生活 |

## 2. 沟通技巧指南

### 2.1 有效倾听

**要点**：
- 放下手机，给予完整注意力
- 不打断妻子表达
- 用眼神和肢体语言回应
- 复述妻子的感受，确认理解

**示例对话**：
> 妻子：今天感觉好累，孕吐反应又严重了。  
> 丈夫：（放下手机，认真看着妻子）辛苦了老婆，复述一下，是孕吐让你很不舒服吗？你想让我帮你做点什么？

### 2.2 表达支持

**要点**：
- 使用"我"语句，表达关心而非指责
- 具体化支持行动
- 避免过度建议，先表达理解

**示例对话**：
> 妻子：我好担心宝宝健康。  
> 丈夫：担心是正常的，我也很关心宝宝。我们一起去做产检，听医生怎么说，好吗？

### 2.3 处理冲突

**要点**：
- 避免在情绪激动时争论
- 使用"我感到...因为..."的表达方式
- 寻求共同解决方案
- 必要时暂停，冷静后再谈

**冲突处理模板**：
```
1. 确认感受："我理解你不舒服/不开心"
2. 说明原因："我感到...是因为..."
3. 提出需求："我希望我们可以..."
4. 共同解决："我们一起想想怎么做好吗？"
```

## 3. 情绪支持要点

### 3.1 常见孕期情绪

| 阶段 | 常见情绪 | 丈夫如何支持 |
|------|----------|--------------|
| 孕早期 | 焦虑、疲惫、情绪波动 | 多陪伴、理解情绪变化 |
| 孕中期 | 安心、期待、敏感 | 鼓励自我关爱、共同期待 |
| 孕晚期 | 焦虑、紧张、期待 | 缓解焦虑、做好分娩准备 |

### 3.2 支持策略

**日常支持**：
- 每天花时间单独交流
- 表达感激与赞美
- 主动承担家务
- 关注妻子的饮食起居

**特殊时刻支持**：
- 产检陪同
- 重要产前课程参加
- 分娩陪产准备
- 产后第一时间陪伴

### 3.3 识别需要专业帮助的信号

丈夫需要关注妻子的情绪状态，以下情况建议寻求专业帮助：
- 持续的情绪低落（超过2周）
- 失眠、食欲显著改变
- 过度的焦虑或恐惧
- 对日常活动失去兴趣

## 4. 家庭支持内容模板

### 4.1 每周支持清单

```
【孕X周 丈夫支持清单】

□ 本周关心要点：[根据孕期阶段]
□ 陪伴产检：是/否（如否，安排其他陪同）
□ 共同活动：[如一起散步、产前课程]
□ 表达感谢：至少1次具体赞美
□ 家务承担：[具体项目]
□ 情感交流：每日至少15分钟
```

### 4.2 关键节点检查

| 节点 | 丈夫任务 |
|------|----------|
| 确认怀孕 | 庆祝、开始学习孕期知识 |
| 第一次产检 | 陪同、了解产检流程 |
| 唐筛/大排畸 | 陪同、关注结果 |
| 孕晚期 | 准备待产包、了解分娩计划 |
| 临产 | 陪产、情绪支持 |
| 产后 | 照顾妻子、承担育儿责任 |

## 5. 跨阶段通用要点

### 5.1 持续行动

- **学习**：主动学习孕期知识，不把所有责任推给妻子
- **沟通**：保持开放、坦诚的沟通
- **参与**：积极参与孕期活动，不是旁观者
- **理解**：理解孕期身体变化带来的情绪波动
- **支持**：在妻子需要时提供支持，不过度干涉

### 5.2 避免行为

- ❌ 不重视妻子的感受
- ❌ 把所有孕期知识学习推给妻子
- ❌ 在妻子不舒服时表现出不耐烦
- ❌ 忽视妻子的情绪变化
- ❌ 做重大决定时不与妻子商量
