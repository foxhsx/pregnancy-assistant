本文档总结了孕期教育网站 MVP 计划中的关键决策，包括 MVP 范围、证据政策、伦理治理等方面的核心决定。

## 1. MVP 范围决策

### 1.1 内容范围

| 决策项 | 决定 | 理由 |
|--------|------|------|
| 孕期阶段 | 孕早期/孕中期/孕晚期三阶段 | 覆盖完整孕期周期 |
| 核心主题 | 饮食安全、孕妈安全、心理健康、家庭支持 | 孕期最关键领域 |
| 语言 | 简体中文 | MVP 阶段聚焦中国大陆 |
| 区域 | 中国大陆 | 目标市场明确 |

### 1.2 功能范围

| 决策项 | 决定 | 理由 |
|--------|------|------|
| 搜索功能 | MVP 包含 | 用户查找内容的基础功能 |
| 用户系统 | MVP 不包含 | 减少复杂性，聚焦内容 |
| 互动功能 | MVP 不包含 | 后续迭代考虑 |
| 电商/支付 | MVP 不包含 | 聚焦教育目标 |

## 2. 证据政策决策

### 2.1 证据等级体系

采用四层证据等级体系：

| 等级 | 定义 | 来源示例 |
|------|------|----------|
| Tier 1 | 最强证据 | WHO、ACOG、CDC、国家卫健委 |
| Tier 2 | 高质量证据 | Cochrane、顶级期刊 RCT |
| Tier 3 | 公共/上下文 | 公开案例、专家共识 |
| Tier 4 | 不推荐 | 未审核博客、非同行评审 |

### 2.2 引用规则

- 核心事实必须引用 Tier 1/2 来源
- 安全/风险声明需标注证据强度
- 多来源冲突需明确标注

## 3. 伦理治理决策

### 3.1 案例 sourcing

| 决策项 | 决定 | 理由 |
|--------|------|------|
| 真实案例 | 可使用 | 提供真实参考价值 |
| 匿名化 | 必须 | 保护隐私 |
| 知情同意 | 必须 | 伦理合规 |
| 医院数据 | 需伦理审批 | 合规要求 |

### 3.2 编辑流程

- 6 步标准流程：选题 → 撰写 → 引用检查 → 内部审核 → 专家复核 → 最终审批
- 医学顾问：可选（后期引入）
- 审核周期：每 6-12 个月
