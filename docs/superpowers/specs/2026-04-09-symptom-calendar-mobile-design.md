# 症状记录页面 - 本周日历移动端交互优化设计

## 1. 背景与目标

当前症状记录页面的「本周日历」在移动端（≤375px）下过于拥挤：7列网格布局每个格子仅约45px宽，需塞入周名、日期、严重程度badge、记录条数4行内容，字号被压缩到10-11px，阅读困难。

**目标**：在移动端采用横向滚动卡片流，借鉴系统日历App的直观交互，单手操作友好，信息层次清晰。

---

## 2. 设计方向

### 核心交互：横向滚动日期卡片

每个日期以 **140px × auto** 的独立卡片展示，7天水平排列、可横向滑动。借鉴系统日历App的直观感。

**卡片布局（从上到下）：**

```
┌──────────────────────┐
│ 周一                 │ ← 顶部左：周名（小字，灰色）
│ 7                    │ ← 中部左：大日期数字（34px，加粗）
│                  ●   │ ← 顶部右：严重程度色圆点（14px）
│ 可记录               │ ← 底部：状态文字（主色或灰色）
└──────────────────────┘
```

**交互状态：**
- **默认**：白底 + 淡阴影
- **选中/激活**：青色边框（2px）+ 淡青背景
- **今日**：橙色描边圆角框

**滚动行为：**
- `overflow-x: auto` + `scroll-snap-type: x mandatory`
- `scroll-snap-align: start` 吸附到每张卡片
- `-webkit-overflow-scrolling: touch` 惯性滚动

## 4. 组件规格

### 4.1 日期卡片 `.sp-day-card`

| 属性 | 值 |
|------|-----|
| 宽度 | `140px`（固定，不压缩） |
| 内边距 | `10px 8px` |
| 圆角 | `20px` |
| 背景 | `#ffffff` |
| 边框 | `1px solid rgba(44,122,123,0.08)` |
| 阴影 | `0 12px 28px rgba(90,67,46,0.08)` |
| scroll-snap | `start` |

**选中态**：
- 边框：`2px solid #2c7a7b`
- 背景：`rgba(44,122,123,0.12)`
- padding 减少 1px 补偿边框

**今日态**：
- `::after` 伪元素，inset 6px，`2px solid #ef8b42`，圆角 16px

**卡片内部：**
- `.day-top`：`flex` + `justify-content: space-between` + `margin-bottom: 20px`
- `.weekday`：14px，灰色
- `.date-num`：34px，800加粗
- `.severity-dot`：14px圆点，颜色映射到严重程度（绿/黄/橙/红）
- `.status`：13px，主色（已有记录）或灰色（可记录）

### 4.2 日历容器 `.sp-week-scroll`

- `display: flex`
- `gap: 8px`
- `overflow-x: auto`
- `scroll-snap-type: x mandatory`
- `scroll-behavior: smooth`
- `padding-bottom: 6px`（给滚动条留空间）
- 滚动条样式：6px高，青色半透明

### 4.3 导航按钮 `.sp-nav-btn`

- 圆形：34px × 34px
- 背景：白色半透明 + 淡边框
- 字号：18px

### 4.4 记录卡片 `.sp-record-item`

保持现有样式，仅在移动端做内边距微调（10px）。

## 5. 响应式策略

| 断点 | 行为 |
|------|------|
| ≥600px（平板+） | 恢复现有网格布局（7列固定宽度） |
| 480-599px（较宽手机） | 卡片宽度 120px，缩小间距 |
| 360-479px（普通手机） | 卡片宽度 140px，横向滚动 |
| <360px（极窄） | 卡片宽度 120px，横向滚动 |

## 7. 技术实现要点

- **滚动吸附**：`scroll-snap-type: x mandatory` + `scroll-snap-align: start`
- **惯性滚动**：`webkitOverflowScrolling: touch`
- **选中态同步**：点击卡片时 `setSelectedDate`，同时更新记录列表筛选
- **今日判断**：`dateStr === formatDate(today)`，`today` 来自 `useMemo(() => new Date(), [])`
- **周导航**：点击 `‹ ›` 调用 `setWeekStart(addDays(weekStart, ±7))`
- **CSS 变量**：使用现有 CSS 变量（`--primary`等），不引入新变量体系
