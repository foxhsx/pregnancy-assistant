# 症状记录功能设计

## 概述

在孕期助手 SPA 中新增症状记录模块，帮助孕妈追踪孕期不适症状，为产检提供参考数据。

## 功能入口

Dashboard 新增「症状记录」入口卡片，显示本周记录数量和整体状态。点击进入独立症状记录页面。

## 症状记录页面

### 顶部状态栏
- 当前孕周 + 孕期阶段标签（孕早期/孕中期/孕晚期）

### 日历视图
- 按周展示，横向滚动切换周
- 每天圆点表示有记录：
  - 颜色深浅 = 平均严重程度（1-5分映射到浅→深）
  - 无记录 = 空心圆点
- 点击日期弹出记录详情

### 记录表单（弹窗/抽屉）
每次记录包含：
- **症状类型**（5大类，每类下有子项）
  - 消化系统：孕吐（晨吐/全天吐）、反胃、食欲不振、食欲亢进
  - 全身症状：疲劳、头晕、头痛、发热
  - 睡眠问题：失眠、嗜睡、夜醒、多梦
  - 身体不适：水肿、抽筋、腰背痛、乳房胀痛
  - 心理情绪：焦虑、情绪波动、注意力不集中
- **严重程度**：1-5 分（😀😃😐😟😣）
- **持续时间**：持续/间歇、时长
- **触发因素**：饮食/运动/情绪/环境/其他（多选）
- **备注**：自由文本

### 趋势页
点击「查看趋势」进入：
- 每个症状类别单独的趋势折线图
- 严重程度平均值、最高值
- 时间范围选择：最近2周/4周/全部
- 可将图表分享或截图给医生

## 数据结构

```js
// localStorage key: 'pregnancy-symptoms'
{
  records: [
    {
      id: 'uuid',
      date: '2026-04-08',
      entries: [
        {
          category: 'digestive', // digestive|systemic|sleep|physical|emotional
          symptom: '孕吐',
          severity: 3, // 1-5
          duration: { type: 'intermittent', hours: 4 },
          triggers: ['smell', '空着肚子'],
          note: '下午吃了油腻食物后加重'
        }
      ]
    }
  ]
}
```

## 依赖

- 复用现有 `usePregnancy` hook 获取孕周信息
- 复用 Dashboard 的 card 样式
- 使用 Chart.js（已有依赖）绑定趋势图
- 日历组件自研，约200行

## 页面路由

与现有页面结构一致，通过 `onNavigate('symptom')` 切换。

## 优先级

1. 症状记录主页面 + 日历视图
2. 记录表单
3. 趋势页
