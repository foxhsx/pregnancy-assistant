# 编辑治理与伦理流程 (Editorial Governance & Ethics)

## 概述

本文档定义了孕期教育网站的内容编辑治理流程和伦理审查要求，确保所有内容科学、准确、合规。

**适用范围**：所有内容生产与发布流程  
**目标**：建立规范的内容治理体系，确保内容质量与合规性

---

## 1. 编辑团队角色

### 1.1 角色定义

| 角色 | 人数 | 主要职责 |
|------|------|----------|
| 内容主编 | 1人 | 总体内容把控、最终审批 |
| 内容编辑 | 2-3人 | 内容撰写、审核、发布 |
| 医学顾问（可选） | 0-1人 | 科学准确性复核 |
| 法务顾问（可选） | 0-1人 | 合规性审核 |

### 1.2 职责矩阵

| 任务 | 内容编辑 | 内容主编 | 医学顾问 | 法务顾问 |
|------|----------|----------|----------|----------|
| 内容撰写 | ✅ | - | - | - |
| 初次审核 | ✅ | - | - | - |
| 科学性审核 | - | - | ✅ | - |
| 合规性审核 | - | - | - | ✅ |
| 最终审批 | - | ✅ | - | - |
| 发布执行 | ✅ | - | - | - |

## 2. 内容生产流程

### 2.1 标准流程（6步法）

```
┌─────────────────────────────────────────────────────────────────┐
│  Step 1: 选题规划                                                 │
│  - 根据内容日历规划内容                                           │
│  - 明确目标受众与主题                                             │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Step 2: 内容撰写                                                 │
│  - 根据模板撰写内容                                               │
│  - 包含引用来源                                                   │
│  - 遵循分类法与标签规范                                           │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Step 3: 引用检查                                                 │
│  - 验证所有引用来源                                               │
│  - 确认证据等级                                                   │
│  - 检查引用格式                                                   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Step 4: 内部审核                                                 │
│  - 内容编辑交叉审核                                               │
│  - 检查准确性、完整性、一致性                                     │
│  - 格式与标签检查                                                 │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Step 5: 专家复核（如适用）                                       │
│  - 医学顾问复核科学性                                             │
│  - 法务顾问复核合规性                                             │
│  - 处理反馈意见                                                   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Step 6: 最终审批与发布                                           │
│  - 内容主编最终审批                                               │
│  - 按计划发布时间发布                                             │
│  - 记录发布信息                                                   │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 各阶段时长要求

| 阶段 | 建议时长 | 最长时长 |
|------|----------|----------|
| 选题规划 | 1-2天 | 3天 |
| 内容撰写 | 2-3天 | 5天 |
| 引用检查 | 0.5天 | 1天 |
| 内部审核 | 1-2天 | 3天 |
| 专家复核 | 2-3天 | 5天 |
| 最终审批 | 0.5天 | 1天 |
| **总计** | **7-13天** | **18天** |

## 3. 伦理审查流程

### 3.1 需要伦理审查的内容

| 内容类型 | 审查要求 |
|----------|----------|
| 真实案例分享 | 知情同意 + 匿名化处理 |
| 敏感健康话题 | 医学顾问审核 |
| 统计数据/研究 | 引用来源验证 |
| 用户生成内容 | 内容审核 |

### 3.2 伦理审查要点

**准确性审查**：
- [ ] 事实陈述准确无误
- [ ] 引用来源可靠
- [ ] 证据等级标注正确
- [ ] 无过时信息

**安全性审查**：
- [ ] 无潜在危害内容
- [ ] 风险提示充分
- [ ] 不提供医疗诊断建议
- [ ] 建议咨询专业医生

**公平性审查**：
- [ ] 无歧视性内容
- [ ] 包容性表达
- [ ] 多元视角呈现

**隐私审查**：
- [ ] 无个人隐私泄露
- [ ] 案例匿名化充分
- [ ] 知情同意完备

## 4. 外部顾问整合

### 4.1 医学顾问职责

**职责范围**：
- 审核内容的科学准确性
- 提供专业建议
- 协助处理争议内容
- 参与内容评审会

**工作方式**：
- 定期审核（每月/每季度）
- 按需审核（紧急内容）
- 参与评审会议

### 4.2 医学顾问准入标准

| 要求 | 说明 |
|------|------|
| 资质 | 妇产科执业医师资格 |
| 经验 | 5年以上临床经验 |
| 认证 | 无医疗事故记录 |

### 4.3 合作协议模板

```
【医学顾问合作协议】

一、服务内容
1. 审核孕期教育内容的科学准确性
2. 提供专业建议与反馈
3. 参与内容评审会议

二、服务时间
- 定期审核：每月X小时
- 按需审核：每周X小时

三、费用与支付
- 月费：XXX元
- 支付时间：每月X日

四、保密义务
- 保护用户隐私信息
- 保护网站内容机密

五、协议期限
- 开始日期：XXXX年XX月XX日
- 结束日期：XXXX年XX月XX日
```

## 5. 内容质量标准

### 5.1 内容检查清单

**撰写阶段**：
- [ ] 主题明确，受众清晰
- [ ] 内容结构清晰，逻辑连贯
- [ ] 引用来源标注完整
- [ ] 遵循分类法与标签规范
- [ ] 字数符合要求

**审核阶段**：
- [ ] 无事实错误
- [ ] 引用来源可验证
- [ ] 证据等级标注正确
- [ ] 语言流畅，无错别字
- [ ] 格式统一规范

**发布阶段**：
- [ ] 通过所有审核
- [ ] 最终审批完成
- [ ] 发布时间确定
- [ ] 相关链接有效

### 5.2 质量问题处理

| 问题等级 | 定义 | 处理方式 |
|----------|------|----------|
| 严重 | 事实错误、健康风险 | 立即撤稿，修订后重新审核 |
| 中等 | 引用错误、格式问题 | 修订后重新审核 |
| 轻微 | 错别字、格式问题 | 直接修订 |

## 6. 更新与维护

### 6.1 内容更新流程

```
发现需要更新的内容
        ↓
评估更新类型（事实/时效/格式）
        ↓
执行更新（编辑/重新撰写）
        ↓
重新走审核流程
        ↓
发布更新版本
        ↓
记录更新日志
```

### 6.2 更新触发条件

- 权威机构发布新指南
- 研究结果重大更新
- 用户反馈问题
- 定期审查发现

### 6.3 版本管理

- 记录每次更新内容、日期、责任人
- 保留历史版本供审计
- 重大变更记录变更原因

## 7. 合规要求

### 7.1 法规遵循

**相关法规**：
- 《中华人民共和国个人信息保护法》
- 《互联网信息服务管理办法》
- 《医疗广告管理办法》
- 《广告法》相关条款

### 7.2 禁止内容

- ❌ 医疗诊断与治疗建议
- ❌ 未经审核的医疗产品推荐
- ❌ 虚假夸大的健康声明
- ❌ 侵犯用户隐私的内容
- ❌ 歧视性或不当内容

### 7.3 免责声明

所有健康相关内容必须包含免责声明：
> 本网站内容仅供参考，不能替代专业医疗建议。如有健康问题，请咨询专业医生。
