# 孕期教育网站证据政策 (Evidence Policy)

## 概述

本文档定义了孕期教育网站 MVP 的证据政策、来源层级、引用模板和治理流程。所有内容必须遵循本政策以确保科学性与可信度。

**目标受众**：中国大陆孕妇（简体中文）  
**适用阶段**：孕早期、孕中期、孕晚期  
**审核周期**：每 6-12 个月正式审查一次

---

## 1. 证据等级体系

### 1.1 Tier 1 - 最强证据（权威指南）

**来源**：
- 全球权威机构：WHO（世界卫生组织）、ACOG（美国妇产科医师协会）、CDC（美国疾控中心）、NICE（英国国家健康与临床优化研究院）
- 中国权威机构：国家卫健委、中华医学会、中国妇产科医师协会

**适用范围**：
- 孕期营养建议
- 分娩方式选择
- 产前检查时间表
- 风险评估与筛查

### 1.2 Tier 2 - 高质量证据

**来源**：
- Cochrane 系统综述
- 发表在顶级期刊的系统综述与随机对照试验（RCT）
- 期刊：Nature Medicine, NEJM, The Lancet, BMJ 等

**适用范围**：
- 具体营养素补充剂效果
- 运动与健康结果
- 心理健康干预效果

### 1.3 Tier 3 - 公共/上下文证据

**来源**：
- 公开的去识别化案例汇总
- 专家共识（需明确标注局限性）
- 新近发表但仍在演变中的研究

**适用范围**：
- 真实案例分享
- 地区性研究结论
- 新兴话题的初步信息

### 1.4 Tier 4 - 不推荐作为主要来源

**来源**：
- 未审核的博客文章
- 非同行评审的教程
- 社交媒体建议

**使用规则**：禁止作为主要引用来源，可作为补充参考但需明确标注来源局限性

## 2. 中国大陆本土证据来源

### 2.1 Tier 1 来源（国内）

| 机构 | 文档类型 | 适用范围 |
|------|----------|----------|
| 国家卫健委 | 孕期保健指南、膳食指南 | 营养建议、产检时间 |
| 中华医学会 | 临床诊疗指南、专家共识 | 疾病管理、分娩 |
| 中国妇产科医师协会 | 临床实践指南 | 产科操作规范 |
| 中国营养学会 | 膳食营养素参考摄入量 | 营养指导 |

### 2.2 引用优先顺序

1. 中国本土权威指南（Tier 1 国内）
2. 国际权威机构指南（Tier 1 全球）
3. Cochrane/系统综述（Tier 2）
4. 高质量期刊研究（Tier 2）
5. 专家共识与案例（Tier 3，需标注局限性）

## 3. 引用模板

### 3.1 标准引用格式

```
[标题]. [来源机构/期刊], [作者（如适用）], [年份]. [DOI/URL]. 最后更新: [YYYY-MM-DD].
```

**示例**：
```
孕期体重管理指南. 国家卫健委, 2022. https://www.nhc.gov.cn/. 最后更新: 2022-03-15.
```

### 3.2 中文内容引用

```
[标题]. [来源], [机构名称], [年份]. [URL].
```

**示例**：
```
妊娠期糖尿病管理. 中华糖尿病杂志, 中华医学会, 2023. https://cmab.org.cn/.
```

### 3.3 英文内容引用

```
[Title]. [Source], [Organization/Authors], [Year]. [DOI/URL].
```

**Example**：
```
WHO Guidelines on Antenatal Care for a Positive Pregnancy Experience. World Health Organization, 2016. https://www.who.int/publications/i/item/9789241549912.
```

## 4. 断言与来源规则

### 4.1 核心事实引用要求

- 每个核心事实必须附带一个或多个引用
- 引用必须可追溯、可验证
- 引用必须来自 Tier 1 或 Tier 2 来源（核心事实）

### 4.2 多来源冲突处理

- 如果存在多个来源， 提供简洁的综合说明
- 明确标注任何冲突点
- 优先采用最新、最权威的来源

### 4.3 安全/风险声明

- 必须标注证据强度：Strong（强）、Moderate（中）、Weak（弱）
- 必须标注风险相关性：High（高）、Medium（中）、Low（低）
- 必须说明适用人群与限制条件

**示例**：
> 叶酸补充可显著降低神经管缺陷风险（证据强度：Strong，风险相关性：High）。建议孕前3个月至孕早期每日补充400μg（来源：WHO, 2015; ACOG, 2020）。

## 5. 治理与工作流程

### 5.1 编辑流程

1. **内容撰写**：作者根据模板撰写内容
2. **引用检查**：必须包含引用检查环节
3. **内部审核**：编辑审核 Tier 1/2 来源的准确性
4. **外部复核**：如有外部医学顾问，进行专家复核
5. **发布**：通过审核后发布

### 5.2 审核责任人

| 角色 | 职责 |
|------|------|
| 内容编辑 | 撰写、引用检查、内部审核 |
| 医学顾问（如有） | 外部复核、科学准确性 |
| 项目负责人 | 最终审批、版本管理 |

### 5.3 更新机制

- **常规更新**：每 6-12 个月进行正式审查
- **触发更新**：权威机构发布新指南时进行紧急更新
- **版本记录**：记录每次更新的内容、日期、责任人

---

## 6. 案例来源与伦理

### 6.1 案例来源规则

- 公开的去识别化案例数据
- 已发表的案例研究（带知情同意）
- 国内医院数据需伦理审批与数据使用协议

### 6.2 匿名化要求

- 移除所有个人身份信息（PII）
- 脱敏处理：姓名、日期、地点、特殊事件
- 记录匿名化过程以供审计

### 6.3 知情同意

- 所有真实案例需获得知情同意
- 记录同意过程与同意内容
- 保存同意文档备查

- [ ] 多来源情况已综合说明

### 7.2 内容审核清单

- [ ] 引用格式正确
- [ ] 证据来源为 Tier 1/2（核心事实）
- [ ] 无过时信息
- [ ] 无未标注的推测
- [ ] 符合中国大陆法规
