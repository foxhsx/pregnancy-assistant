# 孕期教育网站内容分类法 (Content Taxonomy)

## 概述

本文档定义了孕期教育网站 MVP 的内容分类法，涵盖孕早期、孕中期、孕晚期三阶段的核心主题与子主题模板。

**目标受众**：中国大陆孕妇（简体中文）  
**语言**：简体中文  
**区域**：中国大陆

---

## 1. 三阶段核心主题框架

### 1.1 孕早期 (妊娠第1-12周)

| 核心主题 | 子主题 | 说明 |
|---------|--------|------|
| 饮食安全 | 营养补充、叶酸摄入、忌口食物 | 孕早期关键营养素与饮食注意事项 |
| 孕妈安全 | 早孕反应应对、产检安排、风险识别 | 身体变化与安全防护 |
| 心理健康 | 情绪波动、焦虑缓解、角色适应 | 心理调适与支持 |
| 家庭支持 | 丈夫角色、沟通技巧、家庭准备 | 家庭成员支持要点 |

### 1.2 孕中期 (妊娠第13-27周)

| 核心主题 | 子主题 | 说明 |
|---------|--------|------|
| 饮食安全 | 钙铁补充、体重管理、加餐选择 | 孕中期营养重点 |
| 孕妈安全 | 胎动监测、妊娠纹防护、运动建议 | 身体护理与安全 |
| 心理健康 | 情绪稳定、自我关爱、兴趣培养 | 心理健康维护 |
| 家庭支持 | 亲子沟通、育儿知识学习、产前准备 | 家庭准备与支持 |

### 1.3 孕晚期 (妊娠第28-40周)

| 核心主题 | 子主题 | 说明 |
|---------|--------|------|
| 饮食安全 | 分娩能量准备、临产饮食、产后饮食 | 分娩与产后营养 |
| 孕妈安全 | 临产征兆、分娩方式选择、产后护理 | 分娩准备与产后护理 |
| 心理健康 | 产前焦虑、放松技巧、心理准备 | 生产心理准备 |
| 家庭支持 | 陪产准备、新生儿护理分工、月子安排 | 产后支持计划 |

---

## 2. 跨阶段通用要点

以下主题贯穿整个孕期：

| 主题 | 说明 | 相关阶段 |
|------|------|----------|
| 家庭支持 | 丈夫照护、情绪支持、沟通技巧 | 全阶段 |
| 情绪管理 | 焦虑缓解、心理调适、放松技巧 | 全阶段 |
| 沟通技巧 | 医患沟通、家庭沟通、产后沟通 | 全阶段 |
| 运动建议 | 孕期运动、安全指导、禁忌 | 全阶段 |
| 睡眠建议 | 睡眠姿势、失眠应对、休息 | 全阶段 |

## 3. 标签系统

### 3.1 内容标签

| 标签 | 英文 | 说明 |
|------|------|------|
| 饮食 | diet | 饮食与营养相关内容 |
| 安全 | safety | 安全性相关提示 |
| 健康 | health | 健康管理与保健 |
| 心理 | mental | 心理健康与情绪 |
| 家庭 | family | 家庭支持相关内容 |
| 产检 | checkup | 产前检查相关内容 |
| 分娩 | delivery | 分娩相关内容 |
| 产后 | postpartum | 产后护理相关内容 |

### 3.2 阶段标签

| 标签 | 覆盖阶段 |
|------|----------|
| 孕早期 | 妊娠第1-12周 |
| 孕中期 | 妊娠第13-27周 |
| 孕晚期 | 妊娠第28-40周 |
| 全阶段 | 贯穿孕期 |

### 3.3 证据等级标签

| 标签 | 含义 |
|------|------|
| 证据-强 | 来自 WHO、ACOG、CDC、NICE 等权威机构 |
| 证据-中 | 来自系统综述、高质量 RCT |
| 证据-弱 | 来自专家共识、案例研究 |

## 4. 内容模板

### 4.1 阶段页面模板

每个阶段的页面应包含：
1. 阶段概述（妊娠周数范围、特点）
2. 核心主题导航
3. 本阶段重点内容
4. 跨阶段链接
5. 家庭支持要点

### 4.2 主题页面模板

每个主题页面应包含：
1. 主题介绍
2. 关键要点（3-5条）
3. 具体建议/操作步骤
4. 证据来源引用
5. 注意事项/禁忌
6. 相关主题链接

---

## 5. 分类法维护

- **更新频率**：每季度审查一次
- **责任人**：内容编辑团队
- **审核流程**：内部审核 → 外部顾问复核（如需要）
- **版本控制**：记录每次更新内容与日期
